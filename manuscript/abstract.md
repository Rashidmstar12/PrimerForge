# Abstract

Existing polymerase chain reaction (PCR) primer design tools frequently fail in high-throughput applications because they rely on rigid, static thermodynamic heuristics. Consequently, assays suffer from off-target amplification, primer-dimer interference in multiplexed panels, and failure to adapt to local salt chemistry variations. To address these limitations, we introduce PrimerForge, an adaptive molecular engineering platform that integrates nearest-neighbor biophysical modeling with ensembled machine learning. The system combines five Gradient Boosted Decision Tree (GBDT) classifiers and a Deep Multi-Layer Perceptron (MLP) with Platt calibration, regularized via Elastic Weight Consolidation (EWC) to enable lab-adaptive fine-tuning without losing core biophysical knowledge. Multiplex panel design is formulated as an Integer Linear Programming (ILP) problem to guarantee dimer-free combinations. On a curated dataset of 292 empirical primer pairs, PrimerForge achieved a classification performance of 0.986 ROC-AUC on a 20% stratified held-out test split, significantly outperforming the standard biophysical rule-based baseline (0.732 ROC-AUC). Platt calibration reduced probability estimation error to a Brier score of 0.057 compared to 0.271 for the baseline. Furthermore, the ILP optimization engine mathematically ensures 100% dimer-free multiplex sets under user-defined constraints. PrimerForge provides a robust, lab-calibrated molecular design framework that mitigates experimental dropouts. It enables automated, high-throughput assay engineering for complex diagnostic applications.

**Keywords**: PCR Primer Design, Machine Learning, Multiplex PCR, Elastic Weight Consolidation, Integer Linear Programming
